## Method Signatures
{% highlight julia %}
jitter!(ec::EChart) #jitter all series with the same arguments
jitter!(ec::EChart, series::Int) #jitter a single series
{% endhighlight %}

## Optional Arguments
{% highlight julia %}
pctxrange::Real = 0.05
pctyrange::Real = 0
{% endhighlight %}
